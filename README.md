# 🧹 Windows Temp Cleaner

This is a professional Python-based system maintenance tool designed for Windows Server and Desktop environments.

## ✨ Features
- **Admin Privilege Check:** Automatically checks for administrative rights.
- **Python Auto-Install:** Installs Python if it's missing on the host machine.
- **Systematic Logging:** Creates detailed reports for every cleanup process.
- **Safe Deletion:** Skips files that are currently in use by the system.

## 🚀 How to Use
1. Download the files.
2. Run `setup.bat` as Administrator.
3. Check the `Logs` folder for the cleanup report.

## 🛠️ Tech Stack
- Python 3.x
- Batch Scripting
- PyInstaller (for EXE creation)
